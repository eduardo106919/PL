# TPC 04

### Pergunta 01

Considere a seguinte gramática independente de contexto (CFG) com símbolo inicial `S` e terminais `{a, b, c}`.

```
S → aSb | SS | a
```

Quais das seguintes palavras são geradas por esta CFG?

- [x] `aaabb`
- [ ] `ababab`
- [ ] `abaabb`
- [ ] `ababa`

### Pergunta 02

Considere a seguinte gramática independente de contexto (CFG) com símbolo inicial `S` e terminais `{a, b, c}`.

```
S → aSa | bSb | aa | bb
```

Quais das seguintes palavras são geradas por esta CFG?

- [x] `bbaabb`
- [x] `abaaba`
- [ ] `ababa`
- [x] `aa`

### Pergunta 03

Considere a seguinte gramática independente de contexto (CFG) com símbolo inicial `S` e terminais `{a, b, c}`.

```
S → AB | BA
A → aA | a
B → bB | ε
```

Quais das seguintes palavras são geradas por esta CFG?

- [x] `aaaabb`
- [ ] `bbb`
- [x] `aaa`
